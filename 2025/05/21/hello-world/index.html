How to set up a website via hexo blog

2025-05-21
Learning_Notes
874 words
  • 學習日記

Below is a simple guide to make a hexo blog post.
Mile stone 1: Deployed a hexo websitelocally in localhost:4000.
Mile Stone 2: Deployed into github page (Free).
Mile stone 3: Install a custome theme.

Quick Start

1.Setting Up the environments

Install Vscode
Install Git
Install Node.js (v22.15.1)

After installed Git & Node.js, install Hexo.

Remarks: The command sudo is optional, if you use sudo (super user do –computer admin right, it require your computer login password.).

More info: Ref

set up the folder for storing the website

First open your vscode.in the upper toolbar of vscode, open a termianl.Enter following command.

$ cd ./Docuemtn refers to navigate to the Document folder
$ $Hexo init blog can create a default hexo site in “blog” folder
$ cd blog refers to navigate to the blogs
$ hexo s refers to make the local web server up at current folder.

Visit http://localhost:4000/ and you will see your site.
